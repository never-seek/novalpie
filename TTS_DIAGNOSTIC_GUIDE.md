### 可能的问题场景

#### 场景 A：语言包缺失
```
[19:25:30] 📋 系统可用引擎:
[19:25:30]   • com.xiaomi.mibrain.speech (小米语音)
[19:25:30] ✅ 引擎实例创建成功: com.xiaomi.mibrain.speech
[19:25:30] ❌ 语言不支持: LANG_MISSING_DATA (-1)
[19:25:30] 💡 建议：检查系统设置→无障碍→TTS，下载中文语音包
```

**解决方法：**
1. 打开手机设置 → 无障碍 → 文字转语音
2. 点击当前引擎的设置图标
3. 下载中文语音包

#### 场景 B：引擎初始化失败
```
[19:25:30] 📋 系统可用引擎:
[19:25:30]   • (空列表)
[19:25:30] ❌ 引擎实例创建失败(状态:-1)
[19:25:30] 💡 建议：检查系统是否禁用了 TTS 服务
```

**解决方法：**
1. 检查手机是否禁用了 TTS 服务
2. 尝试安装 Google TTS
3. 或重启手机

#### 场景 C：引擎"假读"（流程正常但无声音）
```
[19:25:32] ✅ Speak 调用成功
[19:25:32] ▶️ 引擎开始发声...
[19:25:32] 🏁 引擎发声结束
[19:25:32] 🏁 前端收到 onEnd
```

**特征：** "▶️ 开始发声" 和 "🏁 发声结束" 之间**几乎没有间隔**

**原因：**
- 引擎接受了任务，但没有真正发声
- 可能是音频输出通道问题
- 可能是引擎内部错误（无声模式、被其他应用占用）

**解决方法：**
1. **检查媒体音量**（最重要！）
   - 设置 → 声音与振动 → 确保"媒体音量"已开启
   - **不是铃声音量，不是通知音量！**

2. **检查系统 TTS 设置**
   - 设置 → 无障碍 → 文字转语音
   - 点击"播放示例"按钮
   - 如果示例也没声音，说明系统 TTS 故障

3. **切换引擎**
   - 设置 → 无障碍 → 文字转语音
   - 尝试切换到"Google 文字转语音服务"
   - 或下载其他 TTS 引擎

4. **检查静音模式**
   - 关闭手机静音模式
   - 关闭"勿扰模式"

#### 场景 D：音频焦点获取失败
```
[19:25:32] ⚠️ 音频焦点获取失败(码:0)
[19:25:32] ✅ Speak 调用成功
```

**原因：** 其他应用正在占用音频焦点（如音乐播放器、视频播放器）

**解决方法：**
1. 关闭其他音乐/视频应用
2. 重试听书功能

## 🔍 常见问题诊断

### Q1: 悬浮窗显示 "❌ 语言不支持: LANG_MISSING_DATA (-1)"

**原因：** 系统缺少中文语音包

**解决步骤：**
1. 设置 → 无障碍 → 文字转语音
2. 点击当前引擎（如"小米语音"）的设置图标
3. 找到"语音包"或"语言"设置
4. 下载中文（简体）语音包
5. 重试听书功能

### Q2: 悬浮窗显示 "✅ 语言包加载成功" 但还是没声音

**可能原因：**
1. 媒体音量被静音或调小
2. 手机处于静音模式
3. 引擎内部音频输出问题

**解决步骤：**
1. **检查媒体音量**（最常见）
   - 按音量键 → 展开音量面板 → 调大"媒体"音量
   - 或设置 → 声音 → 调大媒体音量

2. **检查静音模式**
   - 关闭静音开关
   - 关闭勿扰模式

3. **验证系统 TTS**
   - 设置 → 无障碍 → 文字转语音 → 播放示例
   - 如果示例也没声音，说明系统 TTS 故障

### Q3: "▶️ 引擎开始发声" 和 "🏁 引擎发声结束" 几乎同时出现

**原因：** 引擎在"假读"，没有真正发声

**可能原因：**
1. 引擎内部音频路由错误
2. 引擎被其他应用占用
3. 系统音频服务故障

**解决步骤：**
1. 重启手机
2. 清除 TTS 引擎缓存
3. 切换到其他引擎（推荐 Google TTS）
4. 或卸载重装 TTS 引擎

### Q4: 悬浮窗完全不出现

**原因：** JS 注入失败或 WebView 未正确加载

**解决步骤：**
1. 确保网络正常，能访问 novalpie.cc
2. 点击右上角粉色刷新按钮重新加载页面
3. 查看 Logcat 日志：`adb logcat | grep "NovalPie"`

## 💡 强烈建议

### 尝试切换引擎

小米手机默认使用"小米语音引擎"，有时会有兼容性问题。

**推荐切换到 Google TTS：**
1. 打开应用商店，搜索"Google 文字转语音"
2. 安装后，设置 → 无障碍 → 文字转语音
3. 选择"Google 文字转语音服务"作为默认引擎
4. 点击"播放示例"验证是否正常
5. 重试 NovalPie 听书功能

打开：设置 → 无障碍 → 文字转语音

**检查项：**

- [ ] 引擎设置中是否下载了中文语音包
